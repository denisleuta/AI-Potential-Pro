# ⚽ Радарный график характеристик игроков ФК Зенит

Это приложение на Streamlit позволяет визуализировать и сравнивать статистику футболистов академии Зенит с использованием радарных графиков. Вы можете выбрать игрока, метрики для анализа и увидеть, как его показатели соотносятся со средними значениями по команде и его позиции.

## 🚀 Технологический стек

- 🖥️ **UI/UX**: [Streamlit](https://streamlit.io/)
- 📊 **Визуализация**: [Plotly](https://plotly.com/python/)
- 🧮 **Обработка данных**: [Pandas](https://pandas.pydata.org/), [Numpy](https://numpy.org/)
- 🤖 **ML-препроцессинг**: [Scikit-learn](https://scikit-learn.org/)

---

## 🔄 Предобработка данных

### 1️⃣ Очистка и нормализация

✔️ Контекстно-зависимая обработка пустых значений
✔️ Импутация нулями (в колонках, где это нужно) по позициям
✔️ Нормализация с `MinMaxScaler`

```python
scaler = MinMaxscaler()
df[numeric_cols] = scaler.fit_transform(df[numeric_cols])
```

### 2️⃣ Вычисление рейтингов

- Группировка метрик по категориям: **Атака, Защита, Пас и др.**
- Расчет средних значений по каждой категории
- Вычисление общего рейтинга игрока

```python
df["overall_score"] = df[[col for col in df.columns if "_score" in col]].mean(axis=1)
```

## 🛠️ Установка и запуск

1. Убедитесь, что у вас установлен Python 3.8 или выше.
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/mariatrunina/StarUp
   cd StarUp
3. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

4. Запустите приложение:

   ```bash
   streamlit run main.py
   ```
